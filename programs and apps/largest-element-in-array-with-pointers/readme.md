# Largest Element in Array Using Pointers (C++)

This is a short C++ program created as part of a class assignment. The goal of this exercise is to practice accessing array elements using pointers and pointer arithmetic to find the maximum value.

# 📌 Description

The program defines a fixed array of integers and uses a pointer to iterate through each element. It compares each value using pointer arithmetic to determine the largest element in the array, then displays the result.

# 🎯 Objective

- Understand how pointers work in C++
- Learn how to access array elements using pointer arithmetic (*(pointer + i))
- Practice using loops to traverse arrays via pointers
- Reinforce memory and addressing concepts

# ⚙️ How It Works

1. A pointer is initialized to point at the first element of the array
2. The first element is stored as the initial largest value
3. A loop iterates through remaining array elements using pointer arithmetic
4. Each element is compared with the current largest value
5. The largest value is updated if a larger element is found
6. The final largest element is printed to the console

# 📎 Notes

This program is intentionally simple and short since it was created as a classroom exercise focused on a specific concept. The array size is hardcoded to 5 elements.
