# Sum of Two Integers Using Pointers (C++)

This is a short C++ program created as part of a class assignment. The goal of this exercise is to practice using pointers to access integer values and calculate their sum.

# 📌 Description

The program defines two integer variables and uses pointers to read their values. It then adds the values pointed to by the pointers and prints the total.

# 🎯 Objective

- Understand how pointers work in C++
- Learn how to access variable values through pointers (`*pointer`)
- Practice using pointers to reference existing variables
- Reinforce the concept of memory addresses and dereferencing

# ⚙️ How It Works

1. Two integer variables are declared: `a = 5` and `b = 7`
2. Two pointers are initialized to store the addresses of `a` and `b`
3. The pointer values are dereferenced using `*p1` and `*p2`
4. The dereferenced values are added together to compute the sum
5. The final sum is printed to the console using `std::cout`

# 📎 Notes

This program is intentionally simple and short since it was created as a classroom exercise focused on basic pointer usage. The values are hardcoded for demonstration.
