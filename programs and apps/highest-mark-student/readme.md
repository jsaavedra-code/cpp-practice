# Highest Mark Student (C++)

This is a short C++ program created as part of a practice exercise. The goal is to find the student with the highest marks from a fixed array of student records and print their name and score.

# 📌 Description

The program defines a `Student` structure with two fields: `name` and `marks`. It then initializes an array of three students and iterates through the array to determine which student has the highest marks.

# 🎯 Objective

- Practice using C++ structures to group related data
- Learn how to compare values in an array
- Find and print the maximum value using a simple loop
- Reinforce basic `std::cout` output formatting

# ⚙️ How It Works

1. A `Student` struct is defined with `std::string name` and `int marks`
2. An array of `Student` objects is initialized with hardcoded names and marks
3. The program assumes the first student has the highest marks initially
4. It loops through the remaining students and updates the index when a higher mark is found
5. Finally, it prints the top student's name and marks to the console

# 📎 Notes

- The student list is hardcoded for demonstration and includes only three entries
- This example is intended for beginner practice with arrays, structs, and loops in C++
