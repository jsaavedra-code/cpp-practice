# Pointer Array Access (C++)

This is a short C++ program created as part of a class assignment.
The goal of this exercise is to practice accessing array elements using pointers instead of traditional indexing.

## 📌 Description

The program defines a fixed array of integers and uses a pointer to iterate through the array.
It accesses each element using pointer arithmetic and prints the values to the console.

## 🎯 Objective

* Understand how pointers work in C++
* Learn how to access array elements using pointer arithmetic
* Reinforce basic loop and memory concepts

## ⚙️ How It Works

* A pointer is assigned to the first element of the array
* A loop is used to traverse the array
* Each element is accessed using pointer arithmetic (`*(pointer + i)`)
* The values are printed in order

## 📎 Notes

This program is intentionally simple and short since it was created as a classroom exercise focused on a specific concept.
