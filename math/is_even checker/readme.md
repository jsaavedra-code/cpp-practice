# Even or Odd Checker (C++)

This program determines whether a given integer is even or odd. It does exactly that—nothing more, nothing less.

## 📌 Description

The program asks the user for an integer and evaluates whether it is divisible by 2. The logic is handled by a separate function, keeping the code structured and readable instead of dumping everything into `main`.

## 🎯 Objective

* Demonstrate the use of functions in C++
* Apply boolean logic (`true` / `false`)
* Use the modulus operator to evaluate divisibility

## ⚙️ How It Works

* The user enters an integer
* The function `is_even(int n)` checks if the number is divisible by 2
* The program outputs whether the number is even or odd
* No unnecessary drama, just a result

## 🧠 Concepts Used

* Functions
* Boolean values
* Conditional statements (`if-else`)
* Input/output (`cin`, `cout`)
* Modulus operator (`%`)

## 📎 Notes

The program is intentionally straightforward. If it fails, the problem is not complexity.
